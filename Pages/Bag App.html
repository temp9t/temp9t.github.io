Case Study Of Bag App Project

This is a Case Study of Designing an Android Application.
The goal of the project was to develop a simple app User Interface to shop bags with ease. Following you will find all the information regarding this project. The project was concluded in 3 months. This app was design by keeping in the focus that bags should be easily shop and without any hassel and is deliver to our customes at there door step.



Business Goal

To develop an app to where any perosn can easlily search and shop for his/her desired BAG.

Usability Goal

Make the application so easy to use user can shop for BAG and Bag accessories with no hassle.
